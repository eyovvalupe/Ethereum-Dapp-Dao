## 基于 chainlink vrf的 nft盲盒设计
VRF 为链上安全可验证随机数，请查看 [chainlink vrf官方文档](https://docs.chain.link/docs/get-a-random-number);

## 操作步骤 
- 安装依赖
```
npm install 
```

- 测试合约
```
npx hardhat test
```
