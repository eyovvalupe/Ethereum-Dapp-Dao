
## 喂价预言机 
区块链是非常安全可靠的价值交换网络，但却无法安全防篡改地获取链下数据或将数据发送至链下系统。使用 Chainlink 预言机喂价, 通过预言机网络在链上直接获取实时金融市场价格数据

## 测试流程 
### 安装依赖
```
npm install 
```

### 执行测试脚本 
```
npx hardhat run scripts/deploy.js --network kovan
```
