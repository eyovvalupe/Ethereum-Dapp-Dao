## 众筹合约前端
本样例演示众筹合约的基本流程，包括部署合约，合约，启动众筹项目，

## 操作流程
- 配置私钥  
在 .env 中放入的私钥，格式为 
```
VITE_PRIVATE_KEY=xxxxxxxxxxxxxxxx  //私钥
VITE_INFURA_ID=yyyyyyyy  //infura id
VITE_CONTRACT_ADDRESS=zzzzzzzzzzzzz  //合约地址
```
然后代码自动从中读取

- 安装依赖
```
yarn
```

- 启动项目
```
yarn dev
```

- 编译项目
```
yarn build
```
