## 前言
Waffle 是一款适配 ehterjs 的智能合约测试库。本样例演示了使用 Waffle 进行测试的基本流程及使用方法


## 操作步骤  
- 1 安装依赖
```
yarn install
```

- 2 编译合约：
```
yarn build
```

- 3 执行测试  
```
yarn test
```
